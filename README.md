# 🏪 Kiosko - Sistema de Punto de Venta

Sistema completo de punto de venta para kioskos, con panel de administración y carrito de compras.

## ✨ Características

### 👤 Para Clientes
- 🛍️ Catálogo de productos organizado por categorías
- 🔍 Búsqueda de productos en tiempo real
- 🛒 Carrito de compras interactivo
- 📱 Compartir pedido por WhatsApp con formato profesional (fecha, hora, productos, total)

### 👨‍💼 Para Administradores
- 🔐 Login exclusivo con número de teléfono autorizado
- 📊 Dashboard con estadísticas en tiempo real (día, semana, mes)
- 📈 Gráficos de ventas y productos más vendidos
- 📦 Gestión completa de categorías y productos (CRUD)
- ✅ Gestión de pedidos con estados: Pendiente, Completado, Rechazado

## 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura de las páginas |
| CSS3 | Estilos, Flexbox, Grid, Animaciones |
| JavaScript | Lógica de la aplicación |
| Chart.js | Gráficos del dashboard |
| LocalStorage | Persistencia de datos |
| html2pdf | Generación de PDF para pedidos |
